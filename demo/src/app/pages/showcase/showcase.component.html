Getting Started

Quick setup guide for using Finzly components

1. Import Components

All components are available through a single import:

import {{ '{' }} FinzlyCustomButtonComponent, FinzlyInputComponent {{ '}' }} from '@npmswapstech/finzly-theme';

2. Setup Toast Service

Inject FinzlyToastrService in your component to enable toast notifications:

constructor(private toastrService: FinzlyToastrService) {{ '{}' }}

showToast() {{ '{' }}
  this.toastrService.show({{ '{' }}
    type: 'success',
    message: 'Operation completed!'
  {{ '}' }});
{{ '}' }}

3. Use Components

Start building with Finzly components:

<finzly-input
  label="Email"
  placeholder="Enter email"
  [fullWidth]="true">
</finzly-input>

<finzly-custom-button
  variant="primary"
  [fullWidth]="true">
  Submit
</finzly-custom-button>
