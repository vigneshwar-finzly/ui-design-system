Component API

Props / Inputs

Property Type Default Description
type 'solid' | 'soft' | 'outline' 'solid' Badge style type
variant 'primary' | 'success' | 'warning' | 'error' | 'info' | 'neutral' | 'custom' 'primary' Badge color variant
size 'sm' | 'md' | 'lg' 'md' Badge size
pill boolean false Makes the badge fully rounded
dot boolean false Adds a colored dot before text
count boolean false Displays as a small round count badge
customColor string undefined Custom color (hex) for variant="custom"
icon string undefined Icon HTML/SVG string to display before text
iconAfter string undefined Icon HTML/SVG string to display after text
removable boolean false Adds a remove/close button
clickable boolean false Makes the badge clickable with hover effects
